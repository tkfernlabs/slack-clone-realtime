üìã How to Test Audio Calls

  1. Open Two Browser Windows: Use different browsers or incognito mode
  2. Create Two User Accounts:
    • Register User 1 (e.g., alice@example.com)
    • Register User 2 (e.g., bob@example.com)
  3. Join Same Workspace: Both users should be in the same workspace
  4. Start Direct Message:
    • Click on a user in the sidebar to open DM
    • Look for the üìû phone icon in the header
  5. Initiate Call: Click the phone icon to start calling
  6. Accept Call: On the other browser, accept the incoming call
  7. Test Features:
    • Test mute/unmute functionality
    • Check call duration timer
    • End call when finished
